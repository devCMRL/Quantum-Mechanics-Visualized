# Third-party software

| Component | Role in QM Visual |
|-----------|-------------------|
| [Streamlit](https://streamlit.io/) | Application shell and controls |
| [NumPy](https://numpy.org/) | Array numerics |
| [SciPy](https://scipy.org/) | Special functions (e.g. Hermite polynomials, spherical harmonics) |
| [Plotly](https://plotly.com/python/) | Interactive plots |

Core chapters rely on in-repo models in `qm_course/`. Additional solvers or visualization stacks used by future chapters should be recorded in this table with version and license information.
